Lookups

The Lookups options let you change the web sites used for callsign lookups. By default KComm uses HamCall and QRZ.com.

URL 1

The URL specified is used for the first site called to look up a callsign using your default web browser (the two ?'s above the field where the call can be entered in both the main window and the edit dialog.)

The Hint in the field below contains a text descxription that appears when you point at the ?

URL 2

The second URL and its corresponding Hint are used to look up a callsign using the second ?

The callsign to be looked up is appended to the appropriate URL string before it is sent to the browser.
